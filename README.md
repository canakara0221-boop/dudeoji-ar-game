# 🐹 두더지 잡기 — 몸으로 하는 AR 게임 (웹 MVP)

카메라가 아이 **손(몸)을 인식**해서, 튀어나오는 두더지를 손으로 빵! 때리는 게임.
설치·스토어 없이 **브라우저에서 바로** 돕니다. (구글 MoveNet 포즈인식, 무료)

## 핵심
- 단일 파일 `index.html` (외부 라이브러리는 CDN — **인터넷 필요**)
- 카메라는 **https 또는 localhost** 에서만 켜집니다 (브라우저 보안 규칙)

## 1) 내 PC에서 바로 테스트 (제일 쉬움)
PC에 웹캠이 있으면:
1. `serve.bat` 더블클릭 (또는 이 폴더에서 `python -m http.server 8000`)
2. 브라우저에서 **http://localhost:8000** 열기
3. 카메라 허용 → "시작하기" → 한 발 물러서서 손으로 두더지 때리기

## 2) 폰/스마트TV에서 테스트 (실사용 환경)
폰 카메라는 **https** 라야 켜집니다. 무료 호스팅에 올리면 됩니다:
- **Netlify Drop**: https://app.netlify.com/drop 에 이 폴더를 끌어다 놓기 → https 주소 즉시 발급 → 폰에서 열기
- 또는 **GitHub Pages / Vercel** (무료)
- 폰에서 열고 → TV에 **화면 미러링**(아이폰 AirPlay / 안드로이드 스마트뷰)하면 큰 화면

## 게임 동작
- 손목(좌·우) 위치를 추적 → 두더지와 겹치면 점수 +1
- 60초 / 시간 지날수록 두더지 빨라짐 / 끝나면 점수 표시
- 거울 모드(좌우 반전)라 아이가 자연스럽게 봄

## 이건 MVP — 수요 검증용
- 목적: 앱·스토어·규제 투자 *전에* "이게 먹히나"를 공짜로 보기
- 인스타·맘카페에 링크 뿌려 **반응·재방문**부터 측정 → 먹히면 그때 native 앱
